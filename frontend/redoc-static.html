Authentication

Operations related to user authentication.

User login

Authenticates a user with a username and password.

Request Body schema: application/json
required
username
required
string

Username for authentication.

password
required
string

Password for authentication.

Responses

Production server

https://api.domain.example/login

Staging server

https://api-dev.domain.example/login

InteractiveLab

Operations to access and submit data for interactive labs.

Submit interactive lab data

Submit data and results from an interactive lab session.

Authorizations:
BearerAuth
Request Body schema: application/json
required
lab_id
required
string

Unique identifier for the lab.

user_data
required
object

Data input by the user during the lab.

Responses

Production server

https://api.domain.example/interactive-lab

Staging server

https://api-dev.domain.example/interactive-lab

Request samples

  • Payload
Content type
application/json
{
  • "lab_id": "lab001",
  • "user_data": {
    • "experiment_result": "Successful",
    • "parameters": {
      • "temperature": 25,
      • "pressure": 1
      }
    }
}

Response samples

  • 201
  • 401
Content type
application/json
{
  • "message": "Lab data submitted successfully.",
  • "submission_id": "subm456"
}
